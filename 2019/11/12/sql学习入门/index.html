SQL的组成类型

创建表

创建格式：create + tableName + (创建的表字段)；

  • int 数字类型
  • decimal 数字类型(可以设置精确度)
  • timestamp 日期类型(不可为空)
  • datatime 日期类型(可以为空)
  • varchar 字符串类型

增加数据

格式：insert into table_name(字段，字段，字段—) values(参数，参数，参数—)
